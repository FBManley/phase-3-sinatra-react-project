# Usage/Notes:
Fork and clone
run $ bundle install
run $ bundle exec rake db:seed
run $ bundle exec rake server
1. Add a record label: Read and Create
2. Add an artist: Read and Create
3. Update an artist siging: Update
4. Delete an artist: Delete
5. Repeat until full
